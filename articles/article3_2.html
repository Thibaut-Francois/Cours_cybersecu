Articles Root Me : Javascript - Authentification

Publié le 01 juilet 2025

Nous arrivons sur une page contenant un formulaire de login nous demandant un Username et un Password.

Le titre du défi nous indique que ce que nous cherchons est dans le JS (Javascript), nous pouvons donc directement ouvrir le code source avec un simple click droit

La taille de la scroll barre verticale éveille les soupcons. Nous pouvons activer le retour a la ligne automatique pour faire apparaitre ce que l'on essaie de nous cacher.

La lighe de code caché est un lien vers un script js du nom de "login.js"

Une fois le lien clické ont peu tout de suite voir le pseudo et le mote de passe en clair (non crypté) dans le code, c'est gagné !
