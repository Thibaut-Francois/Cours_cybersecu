Ce défi ressemble à s'y méprendre avec le précédent. Regardons directement dans le code source où l'on peut voir que le mot de passe crypté subit deux fois la fonction unescape.

On fait donc passer la variable dans un décodeur unescape.

Et on le fait un deuxième fois avec le résultat précédent.
